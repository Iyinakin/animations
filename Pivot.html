PIVOT ANIMATOR

Pivot animator is a desktop animator used for making animations. there are different general versions of Pivot animator. They are:

  • version 2
  • version 3
  • version 4
  • PIVOT ANIMATOR v5

The animator I am presently using is 'v5'. It is the only version that can have multi-colored stickfigures. The rest only have black and white figures, which you can edit to give different colors in v5 or give one color in the animation. There is a way to link sticknodes and pivot together(i.e. using pivot stickfigures in sticknodes), for this, verion 2 stickfigures will be needed. They are the only stickfigures that can be used in sticknodes (both sticknodes and sticknodes pro).

There are two items (that I know of) that you can use to make animations in Pivot v5. Thy are:

  1. Colorful stickfigures
  2. Sprites (images that you can link together to make a stickfigure)

To learn more about pivot, go to the pivot website. Here you can get new stickfigures from their stk library, get information about the different versions available for download and so on.
